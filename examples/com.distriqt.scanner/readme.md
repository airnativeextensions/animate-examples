Animate CC Example: 

Sample includes UI and functionality to support scanning barcodes using the device camera as well as a custom overlay UI for the scanning interface.

Folder project structure:

ANE (air native extensions) - please download the latest ANE's and palce them here
src (project folder)
- includes (actionscript include files to separate code by function)
- scanner.fla (UI and core project code)
- scanner-app.xml (project export settings for iOS and android)
- Default-XXX.pngs (these images are used to scale the project to the correct resolution for the ios devices and also provide a launch image for the application while it is loading)
- ScannerOverlay Images (These images are used to create the scanner UI for different device resolutions)
